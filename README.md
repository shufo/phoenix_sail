Phoenix Sail provides Docker powered local environment for Phoenix.

## Installation

Add `phoenix_sail` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:phoenix_sail, "~> 0.1.0"}
  ]
end
```

```bash
$ mix deps.get
```

then run below command.

```bash
$ mix sail.install
```
